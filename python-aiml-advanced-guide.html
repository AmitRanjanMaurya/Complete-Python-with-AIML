Core Language Fundamentals
Variables & Data Types
int, float, str, bool, NoneType — type(), isinstance(). Dynamic typing kya hai, type coercion

list, dict, set = mutable. tuple, str, int = immutable. Memory impact samjho — yahi bugs ka root cause hai
Control Flow
if/elif/else, for/while loops, break/continue/pass. Match-case (Python 3.10+)
Functions Deep Dive
*args, **kwargs, default params, keyword-only args. First-class functions, closures, decorators
Comprehensions
List, Dict, Set comprehensions. Generator expressions. Conditional comprehensions. Nested comprehensions
String Manipulation
f-strings, format(), slicing, join/split, regex with re module. String methods library
File I/O & Context Managers
open(), with statement, read/write/append modes. pathlib module. JSON, CSV reading
Exception Handling
try/except/else/finally. Custom exceptions (class MyError(Exception)). raise, assert. Exception hierarchy
Object-Oriented Programming (OOP)
Classes & Objects
__init__, __str__, __repr__. Instance vs class vs static methods. @classmethod, @staticmethod
Inheritance & Polymorphism
Single/Multiple inheritance. super(). MRO (Method Resolution Order). Abstract classes (ABC)
Magic Methods (Dunder)
__len__, __getitem__, __iter__, __next__, __add__, __eq__, __hash__. Protocol-based design
Properties & Descriptors
@property, @setter, @deleter. Encapsulation. Data validation. Descriptor protocol
Dataclasses

Advanced Python Concepts
Decorators & Metaclasses
functools.wraps. Stacked decorators. Class decorators. Metaclass basics (__new__, __init__)
Generators & Iterators
yield keyword. Generator functions vs expressions. send(), throw(). itertools module. Lazy evaluation — ML me bade datasets ke liye critical
Context Managers
__enter__, __exit__. @contextmanager decorator. Resource management. Custom context managers
Type Hints & Annotations
typing module: List, Dict, Optional, Union, Tuple, Callable. TypeVar. Generic types. mypy se static checking
Async/Await (asyncio)
async def, await. asyncio.run(). Concurrent ML inference. aiohttp for async API calls
Memory Management
Reference counting, garbage collector. __slots__ for memory optimization. sys.getsizeof(). Weak references
Modules & Packages
__init__.py, __all__. Relative vs absolute imports. importlib. Virtual environments. pip, poetry
Functional Programming
map(), filter(), reduce(). functools module. Partial functions. Immutability. Pure functions concept
// Example: Decorator + Generator + Type Hints in ML context
from functools import wraps from typing import Generator, Iterator import time def timer(func): """Decorator to time ML training steps""" @wraps(func) def wrapper(*args, **kwargs): start = time.perf_counter() result = func(*args, **kwargs) elapsed = time.perf_counter() - start print(f"⏱ {func.__name__} took {elapsed:.4f}s") return result return wrapper def batch_generator(data: list, batch_size: int) -> Generator: """Lazy batch loading — memory efficient for large datasets""" for i in range(0, len(data), batch_size): yield data[i:i + batch_size] # memory use = 1 batch only @timer def train_epoch(model, dataset: list, batch_size: int = 32) -> float: total_loss = 0.0 for batch in batch_generator(dataset, batch_size): total_loss += model.step(batch) return total_loss / len(dataset)

02
Mathematics & Statistics for ML
Linear Algebra · Calculus · Probability · Statistics · Information Theory
Foundation 5–7 weeks
▶
Linear Algebra
Scalars, Vectors, Matrices, Tensors
Dimensions, shapes. Broadcasting rules. Row vs column vectors. Neural network weights are matrices
Matrix Operations
Addition, multiplication, transpose, inverse. Element-wise (Hadamard) vs dot product. np.dot() vs @
Eigenvalues & Eigenvectors
Av = λv. PCA ka heart yahi hai. Covariance matrix decomposition. numpy.linalg.eig()
SVD (Singular Value Decomposition)
Dimensionality reduction, recommendation systems. A = UΣVᵀ. numpy.linalg.svd()
Norms & Distances
L1 (Manhattan), L2 (Euclidean), L∞ norms. Cosine similarity. Regularization mein directly use
Vector Spaces & Projections
Span, basis, rank, null space. Orthogonality. Gram-Schmidt. Least squares solution
Calculus & Optimization
Derivatives & Partial Derivatives
Chain rule — backpropagation ka core. Partial derivatives: ∂L/∂w. Symbolic vs numerical differentiation
Gradient & Gradient Descent
∇f — direction of steepest ascent. Gradient descent: w = w - α∇L. Learning rate selection
Convexity & Saddle Points
Convex vs non-convex loss landscapes. Local minima, saddle points. Why deep learning is hard to optimize
Taylor Series & Approximations
Second-order optimization methods (Newton's method). Hessian matrix. Adam optimizer intuition
Lagrange Multipliers
Constrained optimization. SVM derivation. Dual problem. KKT conditions
Probability & Statistics
Probability Fundamentals
P(A), P(A∩B), P(A|B). Independence. Conditional probability. Joint vs marginal distributions
Bayes' Theorem
P(A|B) = P(B|A)P(A)/P(B). Prior, likelihood, posterior. Naive Bayes classifier ka foundation
Probability Distributions
Normal (Gaussian), Bernoulli, Binomial, Poisson, Uniform, Exponential. PDF, CDF, sampling
Descriptive Statistics
Mean, median, mode, variance, std dev. Skewness, kurtosis. Percentiles, IQR. Box plots
Hypothesis Testing
Null hypothesis, p-value, significance level. t-test, chi-square test, ANOVA. A/B testing
MLE & MAP Estimation
Maximum Likelihood Estimation. Maximum A Posteriori. Log-likelihood. Connection to loss functions
Information Theory
Entropy H(X) = -Σp(x)log p(x). Cross-entropy loss. KL divergence. Mutual information
Central Limit Theorem
Why sample means are normally distributed. Confidence intervals. Bootstrap sampling

Classical Machine Learning
Scikit-learn · Supervised/Unsupervised · Ensemble Methods · Model Selection
ML Core 7–9 weeks
▶
Supervised Learning — Regression
Linear Regression
OLS, Gradient Descent solution. Cost function (MSE). Closed-form vs iterative. Assumptions: linearity, homoscedasticity, no multicollinearity
Ridge, Lasso, ElasticNet
L2 regularization (Ridge), L1 (Lasso), combined (ElasticNet). Feature selection via Lasso. Cross-validated alpha selection
Polynomial Regression
PolynomialFeatures. Overfitting demonstration. Bias-variance tradeoff visualization. Degree selection via CV
Supervised Learning — Classification
Logistic Regression
Sigmoid function. Binary vs multinomial. Log-loss. Decision boundary. Probability calibration
Support Vector Machines (SVM)
Margin maximization. Support vectors. Kernel trick (RBF, polynomial, linear). C parameter. SVR for regression
Decision Trees
Gini impurity, entropy, information gain. Pruning (max_depth, min_samples). Feature importance. Interpretability
K-Nearest Neighbors (KNN)
Distance metrics (Euclidean, Manhattan, Minkowski). k selection via CV. Curse of dimensionality. KD-tree for efficiency
Naive Bayes
GaussianNB, MultinomialNB, BernoulliNB. Text classification. Laplace smoothing. When it works despite "naive" assumption
Ensemble Methods (Most Important for Industry)
Random Forest
Bagging + feature randomness. OOB error. n_estimators, max_features tuning. Feature importance via impurity. Parallelizable
Gradient Boosting (GBM)
Sequential tree building. Residual fitting. Learning rate, n_estimators, max_depth. sklearn GradientBoostingClassifier
XGBoost
Regularized boosting. xgb.DMatrix. Early stopping. SHAP values. GPU support. Kaggle competitions ka king
LightGBM
Leaf-wise growth (vs level-wise). Faster than XGBoost on large data. Categorical feature handling. lgb.Dataset
CatBoost
Ordered boosting. Native categorical handling without encoding. Best out-of-box performance often
Stacking & Blending
Meta-learner. StackingClassifier in sklearn. Level-1 models → Level-2 meta model. Kaggle winning strategy
Unsupervised Learning
K-Means Clustering
Lloyd's algorithm. Elbow method for k. Inertia vs silhouette score. KMeans++ initialization. MiniBatchKMeans
DBSCAN
Density-based clustering. Noise point detection. eps, min_samples params. Non-spherical clusters. No need to specify k
PCA (Principal Component Analysis)
Variance explained. Scree plot. Dimensionality reduction before modeling. Reconstruction. Kernel PCA
t-SNE & UMAP
Non-linear dimensionality reduction for visualization. t-SNE slow on large data; UMAP faster. Perplexity parameter
Anomaly Detection
IsolationForest, LocalOutlierFactor, EllipticEnvelope. One-class SVM. contamination parameter
Model Evaluation & Selection
Classification Metrics
Accuracy, Precision, Recall, F1, F-beta. ROC-AUC, PR-AUC. Confusion matrix. When to use which metric (imbalanced data)
Regression Metrics
MSE, RMSE, MAE, MAPE, R², Adjusted R². Residual analysis. Heteroscedasticity detection
Cross-Validation
K-Fold, Stratified K-Fold, Leave-One-Out, Time-Series Split. cross_val_score(). StratifiedKFold for imbalanced
Hyperparameter Tuning
GridSearchCV, RandomizedSearchCV, HalvingGridSearchCV. Optuna (Bayesian optimization). param_grid design
Bias-Variance Tradeoff
Underfitting (high bias) vs overfitting (high variance). Learning curves. Validation curves. Regularization effect
Scikit-learn Pipelines
Pipeline() — preprocessing + model chain. ColumnTransformer. FunctionTransformer. No data leakage guarantee
Imbalanced Data Handling
SMOTE, ADASYN (imbalanced-learn). class_weight='balanced'. Oversampling vs undersampling tradeoffs
Model Interpretability
SHAP values (shap library). LIME. Permutation importance. Partial dependence plots. feature_importances_

Interview Prep — Key Questions

Bias-variance tradeoff explain karo
High bias = underfitting (model too simple, misses patterns). High variance = overfitting (memorizes training data, fails on new data). Tradeoff: complex model = low bias, high variance. Solution: regularization, more data, cross-validation. Plot learning curves to diagnose.
Gradient descent ke variants kya hain?
Batch GD: full data per step (slow, accurate). SGD: 1 sample per step (noisy, fast). Mini-batch: compromise (industry standard). Adam = adaptive learning rate + momentum. AdamW adds weight decay. Learning rate warmup critical for transformers.
Random Forest vs XGBoost kab use karein?
Random Forest: parallel training, robust, less tuning, handles missing values. XGBoost: sequential, usually better accuracy, more tuning needed, early stopping available. Practical rule: XGBoost se start karo tabular data pe, agar tuning time nahi to Random Forest.
Cross-entropy loss kya hai aur kyon use hoti hai?
-Σ y_i * log(p_i). Classification ke liye: model ko confident aur correct hone par reward, confident aur wrong hone par heavily penalize. MSE se better kyunki log scale pe large confident mistakes bahut zyada penalize hote hain → faster correct learning.
Imbalanced dataset kaise handle karein?
1) Metrics: F1, AUC-PR (accuracy nahi). 2) class_weight='balanced'. 3) SMOTE oversampling. 4) Threshold tuning. 5) Anomaly detection approach. Pehle try: class_weight + AUC-PR metric. SMOTE is last resort — can introduce noise.
Transformer attention mechanism explain karo
Attention(Q,K,V) = softmax(QKᵀ/√d_k)V. Query: "main kya dhundh raha hoon", Key: "mere paas kya hai", Value: "actual information". Scaled by √d_k to prevent softmax saturation. Multi-head: parallel attention in different representation subspaces.
RAG kya hai aur kab use karein?
Retrieval-Augmented Generation: LLM ko external knowledge dena at inference time. Use when: domain-specific docs hain, real-time info chahiye, hallucination reduce karni hai, full fine-tuning costly ho. Fine-tuning use karo jab: style/behavior change chahiye ya specific task consistently perform karni ho.
Data leakage kya hai aur kaise rokein?
Future/test data ka training me ghusna = inflated metrics jo production pe fail ho. Rokne ke liye: sklearn Pipeline use karo (fit only on train), time-based split for time series, target encoding ke saath CV properly, imputation/scaling train pe fit karo only.
